QUANTIFICATION_OF_QUALIA

Qualia are objects which are observed in at least one subjective vantage. The following definitions attempt to model the way qualia tend to exhibit logically predictable patterns (according to all cumulative observations of such qualia).

WORK = (FORCE / DISTANCE).

WORK := “the amount of energy required to displace an object with mass”.

FORCE = (MASS x ACCELERATION).

FORCE := “the application of energy towards changing the velocity of some object with mass”.

ACCELERATION = (FORCE / MASS).

ACCELERATION := “the change in the velocity of some object with mass within some infinitesimally small interval of time”.

VELOCITY = (DISTANCE / TIME).

VELOCITY := “the displacement of an object with mass during some definite interval of time”.

DISTANCE := “a scalar quantity which represents some natural number of uniform partitions of space arranged in linear succession”.

TIME := “a scalar quantity which represents some natural number of uniform partitions of memory (such that memory is the substrate containing empty space and mass)”.
